# Frontend - Vite React (Lab 3)

Frontend nay duoc xay dung bang Vite + React Router cho 2 chuc nang:

- Dang nhap nhan vien bang `TENDN` + `MATKHAU`
- Quan ly lop hoc theo nhan vien dang nhap (xem tat ca lop, chi sua/xoa lop cua minh)

## 1. Cai dat

```bash
npm install
```

## 2. Cau hinh moi truong

Tao file `.env` tu mau `.env.example`:

```bash
VITE_API_BASE_URL=http://localhost:4000
```

## 3. Chay ung dung

```bash
npm run dev
```

Mo trinh duyet tai `http://localhost:5173`.

## 4. Man hinh da co

- `/` : Dang nhap nhan vien
- `/classes` : Quan ly lop hoc
